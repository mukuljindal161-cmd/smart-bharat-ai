# 🇮🇳 Smart Bharat AI
### AI-Powered Civic Companion

Smart Bharat AI is a modern AI-powered civic platform that helps citizens access government services through an intelligent assistant. It provides a clean, responsive dashboard with authentication and an AI chat interface, making government information more accessible and user-friendly.

## 🚀 Live Demo
🔗 https://smart-bharat-ai-omega.vercel.app

## 📂 GitHub Repository
🔗 https://github.com/mukuljindal161-cmd/smart-bharat-ai

## ✨ Features

- 🔐 Secure Authentication
- 🤖 AI Assistant (Gemini API Ready)
- 📊 Responsive Dashboard
- 🌙 Dark & Light Mode
- 🛡 Protected Routes
- ⚡ Fast and Modern UI
- 📱 Mobile-Friendly Design
- ♻ Reusable React Components

## 🛠 Tech Stack

- React (Vite)
- TypeScript
- Tailwind CSS
- React Router
- React Hook Form
- Zod
- Framer Motion
- Lucide React
- Gemini API (Ready for Integration)

## 📦 Installation

```bash
git clone https://github.com/mukuljindal161-cmd/smart-bharat-ai.git

cd smart-bharat-ai

npm install

npm run dev
```

## 🔑 Environment Variables

Create a `.env` file in the project root.

```env
VITE_GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```
